Modina, C. (2025). "Machine Learning Models for Celestial-Seismic Correlation," in *Phobetron: Biblical Prophecy & Celestial Pattern Detection System*, Version 1.2.0, Zenodo. <https://doi.org/10.5281/zenodo.17558316>

### Astronomical Tracking System

If you use the 3D solar system visualization or hyperbolic orbit solver:

Modina, C. (2025). "Real-Time 3D Solar System with Hyperbolic Orbit Support," in *Phobetron: Biblical Prophecy & Celestial Pattern Detection System*, Version 1.2.0, Zenodo. <https://doi.org/10.5281/zenodo.17558316>
